  NEXT STEPS
================================================================================

1. Clone repository:
   $ git clone https://github.com/marcosayo13/public-finance-data-hub.git
   $ cd public-finance-data-hub

2. Setup environment:
   $ python -m venv venv
   $ source venv/bin/activate
   $ make setup

3. Configure credentials:
   $ bash setup_credentials.sh
   OR manually update .env with your settings

4. Place client_secret.json:
   $ cp ~/Downloads/client_secret_*.json secrets/client_secret.json

5. First authentication:
   $ pfdh auth-google
   (Browser will open - click Allow)

6. Test ingestion:
   $ pfdh ingest --source bcb --from 2024-01-01 --to 2024-12-31

7. Test sync:
   $ pfdh sync-drive --folder-id 1dHJ_r69H_idQyMJE4H60AtVNnDgEOHgF --dry-run

8. Go live:
   $ pfdh sync-drive --folder-id 1dHJ_r69H_idQyMJE4H60AtVNnDgEOHgF

   $ crontab -e
